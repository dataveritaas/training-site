About the Instructor

A data professional with 15+ years of industry experience across domains in building scalable data platforms using various cloud tools.

About the Course

This course is a well-curated modern data engineering training material covering day-to-day projects in companies across various domains. It is intended to train at the industry standard of today using the latest concepts. This course covers basic and advanced data engineering with Azure Cloud.
